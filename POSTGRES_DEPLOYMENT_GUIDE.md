## 🚀 خطوات النشر (10 دقائق)

### الخطوة 1: إعداد PostgreSQL في Vercel (3 دقائق)

1. **اذهب إلى Vercel Dashboard**:

   ```
   https://vercel.com/dashboard
   ```

2. **أنشئ قاعدة بيانات PostgreSQL**:

   - اضغط على "Storage" → "Create Database"
   - اختر "PostgreSQL"
   - اسم قاعدة البيانات: `miladak-production`
   - المنطقة: `US East 1` (الأسرع)

3. **انسخ POSTGRES_URL**:
   - ستحصل على رابط مثل:
   ```
   postgres://default:xxxxx@xxxxx.us-east-1.postgres.vercel-storage.com:5432/verceldb
   ```

### الخطوة 2: تشغيل النشر التلقائي (7 دقائق)

```powershell
# في PowerShell، شغل هذا الأمر مع استبدال POSTGRES_URL بالرابط الحقيقي:
.\deploy-postgres-final.ps1 -PostgresUrl "postgres://default:xxxxx@xxxxx.us-east-1.postgres.vercel-storage.com:5432/verceldb"
```
